# Image Optimizer Web App

Resize images to 4:3, export as WebP, set 150 PPI, and download as a batch ZIP.  
Designed for WordPress or general use. Supports multiple image uploads at once.

---

## Features

- Resize images proportionally to **4:3 ratio** without cropping content  
- Optional **white padding** to maintain 4:3  
- Batch upload and **download all images in a ZIP**  
- Output format: **WebP** (optimized)  
- Set **150 PPI** for print/web  
- Fast processing using parallel threads
